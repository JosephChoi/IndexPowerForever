## 1. 제품 개요

### 무엇을 만드는가

Index Power Forever는 미국 ETF를 선택하면 S&P 500 및 NASDAQ 100 지수와 성과를 비교 분석하여, 지수 투자의 장기 우월성을 시각적으로 증명하는 교육용 웹 서비스다.

회원가입 없이 즉시 사용 가능하며, 티커를 검색하는 순간부터 데이터 기반의 비교 분석이 시작된다.

### 왜 존재하는가

대부분의 액티브 ETF, 테마 ETF, 섹터 ETF는 장기적으로 지수를 이기지 못한다. 그러나 많은 투자자들은 "이 ETF가 좋다던데"라는 막연한 기대로 상품을 선택한다. Index Power Forever는 데이터로 이 환상을 직접 검증하게 하고, 확률적으로 유리한 구조인 인덱스 투자의 우월성을 체험하게 한다.

### 연계 도서: 『이길 수 있는 투자만 하라』와의 연결

이길수 저, 2026년 4월 출간 예정인 『이길 수 있는 투자만 하라』의 핵심 주장을 데이터로 직접 체험하는 인터랙티브 웹 서비스로 기능한다. 책의 각 장(PART)과 웹 서비스 화면이 1:1로 연계된다.

| 책 챕터 | 연계 화면 | 핵심 메시지 |
|---|---|---|
| PART 1~2 | `/insights` | 지수 투자의 과학적 근거 |
| PART 3 (9장, 14장) | `/fee-simulator` | 비용 차이의 복리 효과 |
| PART 3 (13장) | `/timing` | 시장 타이밍 실패의 비용 |
| PART 4 | `/retirement` | 퇴직연금 인덱스 전략 |
| 전 챕터 | `/etf/:ticker` | 개별 ETF의 지수 대비 성과 |

## 2. 대상 사용자

### 사용자 유형 1: 투자 초보자

- **설명**: ETF를 처음 접하거나 투자 경험이 1~3년 미만인 개인 투자자. 뉴스나 유튜브에서 추천받은 ETF를 별다른 비교 없이 매수하는 경향이 있다.
- **주요 목표**: 선택하려는 ETF가 지수 대비 얼마나 잘했는지(혹은 못했는지) 쉽게 확인한다.

### 사용자 유형 2: 장기 투자자 (인덱스 투자 지지자)

- **설명**: 이미 인덱스 투자를 실천 중이거나 관심을 가진 투자자. 자신의 투자 철학을 데이터로 재확인하고, 주변에 공유할 근거를 찾는다.
- **주요 목표**: 인덱스 투자의 장기 우월성을 수치와 차트로 확인하고, 공유 가능한 형태로 얻는다.

### 사용자 유형 3: 퇴직연금 가입자

- **설명**: DC형 퇴직연금을 운용 중이지만 원리금보장형 상품에 묶여 있는 직장인. 인덱스 펀드로 전환했을 때의 기회비용을 알고 싶다.
- **주요 목표**: 원리금보장형 vs 인덱스 ETF의 장기 수익 차이를 직접 시뮬레이션한다.

### 사용자 유형 4: 투자 교육자 / 블로거 / 유튜버

- **설명**: 투자 교육 콘텐츠를 제작하는 크리에이터. 시각 자료와 데이터가 필요하다.
- **주요 목표**: 특정 ETF의 지수 대비 성과 데이터와 차트를 콘텐츠 제작에 활용한다.

### 사용자 유형 5: 책 독자

- **설명**: 『이길 수 있는 투자만 하라』를 읽은 독자. 책에서 언급된 데이터를 직접 체험하고 싶다.
- **주요 목표**: 책의 각 챕터에서 설명된 개념을 인터랙티브하게 탐색한다.

## 3. 핵심 사용자 시나리오

### 시나리오 1: ETF 지수 비교 탐색

- **행위자**: 투자 초보자
- **목표**: 유튜브에서 추천받은 ARKK ETF가 실제로 얼마나 좋은 성과를 냈는지 확인한다.
- **기본 흐름**:
  1. 홈 화면에서 "ARKK"를 검색창에 입력한다.
  2. 자동완성에서 "ARKK - ARK Innovation ETF"를 선택한다.
  3. ETF 상세 화면이 열리며 기본 기간(5Y) 수익률 비교 차트가 표시된다.
  4. 차트에서 ARKK(파란선), S&P 500(녹색 점선), NASDAQ 100(주황 점선)을 비교한다.
  5. 초과수익률 차트에서 ARKK가 지수를 이긴 구간(녹색)과 진 구간(적색)을 확인한다.
  6. 성과 지표 테이블에서 CAGR, MDD, 샤프 비율을 비교한다.

### 시나리오 2: 인기 프리셋으로 즉시 탐색

- **행위자**: 장기 투자자
- **목표**: 대표적인 테마 ETF들이 지수 대비 어떤 성과를 냈는지 빠르게 훑어본다.
- **기본 흐름**:
  1. 홈 화면의 인기 프리셋 카드(TQQQ, VYM, GLD, XLK 등)를 클릭한다.
  2. ETF 상세 화면에서 해당 ETF의 지수 대비 성과를 확인한다.
  3. "이김/짐 분석" 탭으로 전환하여 연도별 승패 차트를 확인한다.
  4. 롤링 승률(1Y/3Y/5Y)을 통해 보유 기간별 지수 대비 이길 확률을 확인한다.

### 시나리오 3: 비용 시뮬레이션

- **행위자**: 투자 초보자 / 장기 투자자
- **목표**: 연간 0.1%의 인덱스 ETF와 1.5%의 액티브 ETF의 30년 후 자산 차이를 확인한다.
- **기본 흐름**:
  1. 네비게이션 또는 홈의 "비용 시뮬레이터" 링크를 클릭한다.
  2. 초기 투자금(1억원), 연평균 수익률(7%), 투자 기간(30년)을 입력한다.
  3. 비용 슬라이더로 3가지 시나리오(0.1% / 1.0% / 2.0%)를 비교한다.
  4. 차트와 최종 자산 금액 테이블이 즉시 업데이트된다.
  5. 비용 차이로 인한 30년 후 자산 격차를 확인한다.

### 시나리오 4: 퇴직연금 전략 비교

- **행위자**: 퇴직연금 가입자
- **목표**: 현재 원리금보장형(연 2.5%)으로 운용 중인 퇴직연금을 인덱스 펀드로 전환하면 20년 후 얼마나 더 받을 수 있는지 확인한다.
- **기본 흐름**:
  1. 네비게이션에서 "퇴직연금 시뮬레이터"를 클릭한다.
  2. 현재 적립금(5,000만원), 월 납입액(50만원), 예상 운용 기간(20년)을 입력한다.
  3. 3가지 시나리오(원리금보장 2.5% / S&P 500 인덱스 / NASDAQ 100 인덱스)의 예상 결과를 비교한다.
  4. 각 시나리오의 20년 후 총 자산과 수익 차이를 확인한다.

### 시나리오 5: 타이밍 실패 비용 확인

- **행위자**: 장기 투자자 / 책 독자
- **목표**: S&P 500에서 상위 20일을 놓쳤을 때 수익률이 얼마나 낮아지는지 확인한다.
- **기본 흐름**:
  1. 네비게이션에서 "타이밍 시뮬레이터"를 클릭한다.
  2. 기간(예: 2000~2024), 놓친 상위 상승일 수(0일 / 10일 / 20일 / 30일)를 선택한다.
  3. 각 시나리오별 최종 수익률과 연환산 수익률(CAGR)이 계산되어 표시된다.
  4. "시장에 항상 머물러 있는 것"의 중요성을 수치로 체험한다.

### 시나리오 6: 기간별 ETF 랭킹 탐색

- **행위자**: 투자 교육자 / 투자 초보자
- **목표**: 최근 3년 동안 S&P 500을 가장 크게 이긴 ETF와 가장 크게 진 ETF를 확인한다.
- **기본 흐름**:
  1. 네비게이션에서 "랭킹"을 클릭한다.
  2. 기간 선택 탭(1Y / 3Y / 5Y / 10Y)에서 "3Y"를 선택한다.
  3. 벤치마크(S&P 500 / NASDAQ 100)를 선택한다.
  4. 초과수익률 기준으로 정렬된 ETF 목록을 확인한다.
  5. 특정 ETF를 클릭하면 ETF 상세 화면으로 이동한다.

### 시나리오 7: 책 연계 인사이트 탐색

- **행위자**: 책 독자
- **목표**: 『이길 수 있는 투자만 하라』의 챕터별 핵심 내용을 인터랙티브하게 체험한다.
- **기본 흐름**:
  1. 홈의 책 소개 섹션 또는 네비게이션의 "인사이트"를 클릭한다.
  2. PART별(PART 1~4) 카드 목록에서 관심 주제를 선택한다.
  3. 각 인사이트 카드에서 핵심 개념 설명과 연계 시뮬레이터 링크를 확인한다.
  4. "직접 확인하기" 버튼으로 관련 시뮬레이터 화면으로 이동한다.
